# Big-Data-in-Finance
 《大数据金融》课程的授课课件（含作业）
版本号：V1.02
更新日期：2023.5.9

## 本课件的特点
- 课上练习多，且容易上手。每一个知识点均搭配一个练习，帮助学生迅速巩固对于知识点的理解。
- 课后作业场景丰富，具有挑战性。增加了较多商业场景，避免编程知识和商业场景脱离。
- 图片多，示例多。大部分知识点应用可视化的方式，降低学习难度。
